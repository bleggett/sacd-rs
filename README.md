# SACD ripper client

Designed to be used with a jailbroken SACD player, such as specific models of PS3s or some Sony Bluray players,
which can support running a server to stream raw SACD image data over the network to a client.

This repo currently only implements a client that can connect to the jailbroken device and stream the complete SACD iso to a local PC.

Eventually, it might be a full Scarlet Book rust library impl + a client.

See https://github.com/EuFlo/sacd-ripper and others
